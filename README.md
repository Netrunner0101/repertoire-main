# Petit Projet Personnel
## Projet repertoire 

Petit projet portofolio au travers lequel, j'ai créer un petit répertoire pour enregistrer des personnes .

Réaliser avec :
  - Backend : Spring Boot 
  - Front-end : React , React Material UI et React Router Dom 
  - DB : Postgresql
